Pasos Ejectuar

Paso 1:
    Abra la terminal dentro de Death-Match-Sorting-Data-Structure- y escriba lo siguente:

    - make

Paso 2:
    En la terminal ingrese el ejecutable resultante y ponga el archivo .bin 

    - ./sortingtest Nombre.bin

    Si no tiene archivos con pyhton3 puede ejecutar experimento.py para generar archivos de 1, 10, 50, 100 Y 250 MB de arreglos con los siguentes ordenes Ascendente, Descendente, Random, Parcial(mitad ordenado mitad desordenado),        
    Duplicados.

    - python3 experimento.py
